Pour que le framework fonctionne, les répertoires présents à l'intérieur
du répertoire "Libs" doivent être copiés à l'intérieur du répertoire:
C:\Python27\Lib

Il est important de noter que la librairie "pyside" doit aussi être intallée
sur le système.

De plus, pour que le référentiel local et la fonction "Auto prioritize" fonctionnent, 
vous devez télécharger le projet codesearch (voir répertoire CodeSearch64Bit) sur le site sharepoint
et configurer la variable de session suivante:
CSEARCHINDEX
c:\CheminVersIndex\FichierIndex
Par la suite, un référentiel local doit être créer. Vous pouvez suivre les instructions de 
code search afin de procéder à la création de ce référentiel.

Le fichier de configuration binSrcr.config doit aussi être adapté à votre
système (rechercher la valeur YOURUSER à l'intérieur de ce fichier). Veuillez noter
qu'il est possible que d'autres ajustements soient requis à l'intérieur de ce fichier.
